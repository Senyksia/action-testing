# Infinite Friends
A [BepInEx](https://docs.bepinex.dev/master/articles/index.html) plugin for the game [SpiderHeck](https://store.steampowered.com/app/1329500/SpiderHeck/), to extend the game's multiplayer beyond 4 players.
